# Security System - PCB School

A web-based security registration and incident reporting system for PCB School.

## Features

- **Guest Registration**: Register visitors with name, type, and action
- **Problem Reporting**: Report security incidents with photos
- **Admin Panel**: View and manage all entries and problems
- **Dark Mode**: Toggle between light and dark themes
- **Multi-language Support**: Spanish and English
- **Export to CSV**: Export entries and problems data
- **Responsive Design**: Works on desktop and mobile devices

## Admin Access

To access the admin panel, click the "Admin" button and enter the password.

## Deployment

This site is designed to be deployed on GitHub Pages. Simply push to a GitHub repository and enable GitHub Pages in the repository settings.

## Technologies Used

- HTML5
- CSS3 (with CSS Variables)
- Vanilla JavaScript
- Font Awesome Icons
- Local Storage for data persistence
